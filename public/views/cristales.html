Copa Martini

La copa cóctel o copa martini es el tipo de cristalería para coctelería que se utiliza para elaborar cócteles como el martini, manhattan, daiquiri, y otros que se sirven muy fríos. ¿Ya la reconociste? Seguro que sí. Además, está diseñada para bebidas que no requieren hielo, por lo que la mezcla debe ser enfriada previamente con la ayuda de un shaker. Por otro lado, debes tener en cuenta que este tipo de copas para bebidas tienen una capacidad aproximada de 150 a 200 ml.

Copa Margarita

Como bien lo dice su nombre, esta cristalería para cócteles se utiliza para servir el cóctel margarita. Pero también se puede emplear para otros cócteles, como el Daiquiri. Según el portal de OTC GROUP, puedes decorar tu cócteles con diversas frutas, hielos de frutos congelados, especias, flores y, como es típico de la margarita, puedes endulzar el borde de este tipo de cristalería para coctelería.

Vaso highball o tumbler

Probablemente es uno los vasos para coctelería más utilizados en los bares, ya que su estructura es alta y estrecha, lo que lo hace ideal para tragos largos que requieren mucho hielo, como el cuba libre o el chilcano. Usualmente, estos vasos para trago tienen una capacidad de 300 ml, pero esta medida puede variar.

Vaso old fashioned

También es conocido como rock glass por su uso para el cóctel “on the rocks”. Esta cristalería para coctelería es ideal para servir whiskey, bourbon y diferentes tragos que requieren gran cantidad de hielo. En cuanto a su estructura, este vaso para la coctelería es corto y ancho. Además, suele ser pesado y de material resistente. Su capacidad se encuentra alrededor de los 250 ml. Es, definitivamente, unos de los vasos para tragos más clásicos y elegantes de la historia.

Copa champagne o pompadour

Siempre hay razones para celebrar. Por ello, la copa champagne es el tipo de cristalería y utensilios de bar que no deben faltar en tu casa. La copa pompadour también se utiliza para servir cócteles como daiquiris y otros tragos con hielo molido. Este tipo de copa se caracteriza por tener un diámetro grande y abierto, lo que permite beber con mayor rapidez. Además, al ser ligeramente cerrada por los bordes, se evita que se derrame el contenido ante cualquier movimiento. La capacidad de este tipo de cristalería para coctelería gira en torno de los 150 a 200 ml.

Copa Huracan

La copa hurricane cuenta con una base ancha y un cuello corto y redondo. Es un vaso de trago amplio con una capacidad de alrededor de 400 ml. Es muy recomendado para cócteles de frutas, que requieren una decoración más elaborada.

Copa flauta

Este tipo de copa se usa también para servir el champagne, ya que su forma evita la pérdida rápida del gas. Además, es perfecta para cavas y espumosos. Su capacidad es similar a la copa champagne, entre 150 a 200 ml. Es uno de los tipos de copas y vasos para bar que no deben faltar en casa

Vaso de shot

Este tipo de vaso para trago, también llamado “caballito”, es ideal para pequeñas cantidades de licor, por lo que suele tomarse su contenido en un solo trago. Se utiliza popularmente para beber tequila.
